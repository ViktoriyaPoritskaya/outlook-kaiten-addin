Kaiten ↔ Outlook

Подключение к Kaiten

Адрес вашего Kaiten без слэша на конце. Не попадает в публичные файлы — хранится в вашем профиле Outlook.

Из URL вашего Kaiten: /space/ID/boards. Не попадает в публичные файлы.

Сгенерируй токен в Kaiten: правый верхний угол → Профиль → Доступ к API → Создать токен. Хранится локально в твоём почтовом профиле, на хостинге расширения его нет.

Список подтягивается из пространства, указанного выше.

Что попадает в карточку:
• Тема письма → название задачи
• Тело письма → описание (Markdown)
• Отправитель, дата, список вложений — в начало описания
• Ссылка на исходное письмо — как external link на карточке
